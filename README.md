# Project-Pemrograman-Dasar
Aplikasi Toko Online “Bukan Orang Jawa” merupakan program sederhana berbasis bahasa pemrograman C++ yang dikembangkan sebagai bagian dari proyek Pemrograman Dasar. Aplikasi ini menggunakan antarmuka berbasis console dan menerapkan konsep pemrograman berorientasi objek (OOP) dalam pengelolaan data produk dan transaksi pembelian. Sistem menyediakan dua peran utama, yaitu Admin yang bertugas mengelola produk (tambah, tampil, edit, hapus, dan simpan data ke file) serta Konsumen yang dapat melihat produk dan melakukan pembelian. Data produk dapat dimuat dari file dan disimpan kembali untuk penggunaan berikutnya. Proyek ini bertujuan untuk melatih kemampuan dasar pemrograman, pemodelan sistem dengan UML, serta penerapan OOP dalam C++.
